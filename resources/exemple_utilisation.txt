EXEMPLES D'UTILISATION
par L.RIBEIRO GOMES, K.NICOLLE, E.LE COANT et F.BENMAHDJOUB

    Vous trouverez ci-dessous une suite d'exemples reprenant les différents cas d'utilisation de notre interpréteur BASISTIC.

I. INTERFACE

1. La zone de texte : permet de saisir des instructions en langage BASIC augmenté.
2. Import de fichier : permet d'importer un fichier contenant le code à interpréter
3. Run : permet de lancer l'interprétation
4. Interface graphique : affiche les résultats graphiques de l'interprétation
5. Console : affiche les résultats textuels de l'interprétation


II. PREMIER PROGRAMME

Dans l'éditeur, entrez le texte suivant : DRAWCIRCLE(200,220,50) .
Cette ligne de code signifie que vous allez dessiner un cercle dont le centre sera à 200 pixels du bord gauche, 220 pixels du bord haut et qui aura un rayon de 50 pixels.
Si vous avez saisi cette ligne de code correctement, vous allez voir un cercle apparaître dans l'interface graphique après avoir cliquer sur le bouton "run".

Si vous ne l'avez pas saisi correctement, une erreur devrait apparaître dans la console. Si cela arrive, vérifiez que vous avez parfaitement recopié l'exemple : 
les nombres devraient être compris entre deux parenthèses et avoir une virgule entre chacun d'entre eux.

Il faut être très rigoureux vis-à-vis de la syntaxe, une simple erreur de ponctuation peut empêcher le bon fonctionnement du programme.
